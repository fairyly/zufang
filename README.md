# doubanzufang

> 豆瓣租房抓取

个人项目。仅开源配置,具体代码完善后决定是否开源

数据如何判断更新抓取?

1. 发表日期、回复日期

2. 定时抓取(大概15分钟一次),定时更新

提供微信公众平台私有api:自动回复某个用户。(模拟登陆。时间2天)

主要用来抓取数据进行测试。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
